# 无尽跃迁 (Endless Warptorio)

建好工厂，跃迁奔向下一个星系，一局接一局，永远有新地图。

Build up your factory, warp to a fresh star system, repeat. Every run is a new map. *(English below)*

---

## 核心玩法

- **跃迁倒计时**：倒计时归零就跃迁去新世界。研究科技延长倒计时，发射火箭、投票缩短倒计时。
- **一切归零**：每次跃迁抛弃整个旧星系，工厂、科技、矿物全部留下，所有星球重新生成。
- **永久进度**：跃迁时背包里的科技瓶变成经验，等级越高开局物资越多；手搓、移动、挖矿越练越快，跨世界保留。
- **职业系统**：多种专精职业决定开局白送什么、练什么瓶，随时切换。
- **飞船有限命数**：太空平台跟随你跨越星系，命数用完即报废。
- **世界变体**：每轮每星球随机滚地形、据点、资源等变化，反复跃迁不重样。
- **双货币**：金币买装备，星星投票、续时间。

## 使用方式

安装本模组（需要 Factorio 2.0 + Space Age DLC），新游戏 → 场景 → 选择本场景。

## 使用建议

- **建议用于多人长期服务器，不推荐单人游戏**：各机制都按多人在线设计。
- **这是场景，不依赖模组运行**：开图时脚本已写进存档，服务器和玩家都不需要装本模组，卸载也不影响已有存档。

# English

A custom multiplayer scenario for Factorio 2.0 + Space Age: build up, warp away, start over, forever.

## Core Loop

- **Warp countdown**: when the timer hits zero, everyone warps to a brand-new star system. Research extends the timer; rockets and votes shorten it.
- **Total reset**: every warp abandons the old star system entirely — factory, research and ores are left behind, all planets regenerate.
- **Permanent progression**: science packs in your inventory convert to XP at warp, granting bigger starting kits; handcrafting, running and mining get permanently faster, carried across worlds.
- **Class system**: specialist classes decide your starting kit and what you grind. Switch any time.
- **Ships with limited lives**: space platforms follow you across systems and are scrapped when their lives run out.
- **World variants**: every planet, every run, randomly rolls terrain, outposts, resources and more. Never the same twice.
- **Dual currency**: coins for gear, stars for votes and extending the countdown.

## How to Use

Install this mod (requires Factorio 2.0 + Space Age DLC), then New Game → Scenarios → pick this scenario.

## Recommendations

- **Designed for long-running multiplayer servers; not recommended for single player.**
- **It's a scenario — saves don't depend on the mod.** Scripts are baked into the save at map creation; neither the server nor players need the mod installed, and uninstalling won't break existing saves.
